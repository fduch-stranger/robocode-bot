# Plans

This folder keeps research and tuning plans that are too detailed for bot
README files.

| Plan | Scope |
| --- | --- |
| [Confidence-calibrated virtual selector](confidence-calibrated-virtual-selector-plan.md) | Selector-level calibration plan that maps virtual scores to real hit probability before switching guns. |
| [Displacement gun PIF improvement](gun-displacement-pif-improvement.md) | Plan to replace simple displacement averaging with rotation-normalized play-it-forward replay. |
| [Dynamic cluster aim refinement](dynamic-cluster-aim-refinement.md) | Plan to improve KNN aim extraction after richer fire-context features are available. |
| [Shared gun fire-context features](shared-gun-fire-context-features.md) | Plan to enrich fire-time sample context for KNN, GF, anti-surfer, displacement, and selector confidence. |
